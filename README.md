## Setup
Pre requirement pnpm https://pnpm.io/installation
To install the dependencies:

````bash
# npm
pnpm install

## Development Server

Start the development server on http://localhost:3002

```bash
pnpm run dev
````

## Production

Build the application for production:

```bash
pnpm run build
```

Locally preview production build:

```bash
pnpm start
```
